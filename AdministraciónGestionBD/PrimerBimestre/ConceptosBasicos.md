# Teoría de Bases de Datos

## ¿Qué es una Base de Datos?
Una base de datos es un conjunto organizado de datos que permite almacenar, gestionar y recuperar información de forma eficiente.

## DBMS (Sistema de Gestión de Bases de Datos)
Es el software que permite trabajar con bases de datos.

Funciones:
- Crear bases de datos
- Insertar, modificar y eliminar datos
- Gestionar usuarios
- Garantizar seguridad

Ejemplos:
- MySQL
- PostgreSQL
- MongoDB

## Tipos de Bases de Datos

### Relacionales
- Usan tablas
- Estructura fija
- Lenguaje SQL

### No Relacionales (NoSQL)
- Estructura flexible
- Escalables
- Tipos:
  - Documentales
  - Clave-valor
  - Grafos

## Características principales
- Persistencia
- Integridad
- Seguridad
- Acceso concurrente

## Administración de BD
Incluye:
- Backups
- Recuperación de datos
- Optimización
- Control de acceso

## Seguridad
- Autenticación
- Autorización
- Encriptación
