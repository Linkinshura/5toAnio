# Diccionario de Datos(Simple)

## ¿Para que sirve? 
Para mantener un orden a la hora de realizar un MER/DER, generando una tabla donde se añade la informacion de cada atributo de una tabla en este orden:

### Columnas:

Ñ°: Numero de atributo (ej 1)

Nombre: Nombre del atributo(ej ID)

Descripcion: Una breve descripcion del atributo (ej Identificador Unico)

Tipo: Tipo de dato del atributo (ej INT)

Longitud: Longitud de dominio del atributo (ej 15)

PK/FK: Indica si es Primary Key, Foreign Key o no (PK, FK, - )

Enlace: En caso de FK, se indica el nombre de la tabla a la que hace relacion
