# 1. Fundamentos de programación gráfica

Este primer tema introduce las bases para trabajar con programación visual en Processing. El objetivo principal es entender cómo se construyen imágenes en pantalla a partir de código.

Se aprende a utilizar la estructura básica del programa (`setup()` y `draw()`), que permite inicializar el entorno y ejecutar instrucciones de forma continua. A partir de esto, se comienza a trabajar con el sistema de coordenadas (x, y), donde cada elemento gráfico se posiciona en la pantalla.

También se introducen las figuras geométricas básicas como rectángulos, elipses, líneas y puntos, junto con el manejo de color mediante funciones como `fill()` y `stroke()`. Estos elementos permiten construir composiciones visuales simples.

Además, se incorporan las primeras nociones de interacción, utilizando el mouse como entrada (`mouseX`, `mouseY`, `mousePressed`), lo que permite que el programa responda a acciones del usuario.

Finalmente, se trabajan funciones simples para agrupar instrucciones de dibujo, dando los primeros pasos hacia la organización del código.
