# Capa de Enlace de Datos - Modelo OSI

La capa de enlace de datos es la capa 2 del modelo OSI y se encarga de la 
transmisión confiable de datos entre dispositivos dentro de la misma red local.

## Funciones principales
- Direccionamiento físico (MAC)
- Encapsulamiento en tramas
- Detección de errores
- Control de acceso al medio
- Control de flujo local

## Subcapas
- LLC: control lógico de enlace
- MAC: acceso al medio y direcciones físicas

## Dispositivos
- Switches
- Tarjetas de red (NIC)

## Ejemplo
En una red local, los dispositivos se comunican usando direcciones MAC 
y verifican errores en las tramas enviadas.

## Característica clave
Permite la comunicación confiable dentro de una misma red física.
