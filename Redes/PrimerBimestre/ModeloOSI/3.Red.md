# Capa de Red - Modelo OSI

La capa de red es la capa 3 del modelo OSI y se encarga de transportar 
los datos entre diferentes redes mediante direccionamiento lógico y enrutamiento.

## Funciones principales
- Direccionamiento lógico (IP)
- Enrutamiento de datos
- Encapsulamiento en paquetes
- Fragmentación de datos

## Protocolos
- IP: direccionamiento de dispositivos
- ICMP: diagnóstico y errores
- Protocolos de enrutamiento (RIP, OSPF)

## Ejemplo
Al enviar datos a otra red, los paquetes viajan a través de distintos routers 
que determinan la mejor ruta hasta el destino.

## Característica clave
Permite la comunicación entre redes distintas, definiendo rutas y direcciones IP.
