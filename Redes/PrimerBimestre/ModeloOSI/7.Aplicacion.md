# Capa de Aplicación - Modelo OSI

La capa de aplicación es la capa 7 del modelo OSI y es la más cercana al usuario. 
Su función principal es permitir que las aplicaciones (como navegadores o clientes de correo) 
puedan comunicarse a través de la red.

## Funciones principales
- Proporcionar interfaz entre aplicaciones y la red
- Identificar usuarios y recursos
- Verificar disponibilidad de servicios
- Gestionar autenticación y acceso

## Protocolos importantes
- HTTP / HTTPS: navegación web
- FTP: transferencia de archivos
- SMTP: envío de correos
- POP3 / IMAP: recepción de correos
- DNS: resolución de nombres a IP

## Ejemplo
Cuando un usuario accede a una web, el protocolo HTTP/HTTPS permite solicitar 
y recibir la información desde un servidor.

## Característica clave
No maneja transmisión física de datos, sino la interacción directa con el software del usuario.
